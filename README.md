# Peter Fu Portfolio

## Setup:
1. git clone https://github.com/peterfuwork/gulp-project
2. cd portfolio
3. npm install
4. npm start
5. Go to localhost:8000 in your browser

## Description:
This is Peter Fu's first npm gulp portfolio. Please go visit http://chiaen.fu.surge.sh/ to see my website online.
